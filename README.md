#SpringCloud-KAB
##描述
	`此项目使用SpringCloud搭建，使用reids引入kab-redis即可，数据源使用kab-datasource`
	`所有的公共配置文件放在github上，配置了手动刷新，访问/actuator/refresh接口即可`

### kab-eureka
`注册中心`

### kab-config
`配置中心，在此配置github路径`

### kab-resource
`服务器资源路径`

### kab-zull
`网关`

### kab-common
`服务使用的Utils、Bean....`

### kab-consultation-*
`资讯`

### kab-course-*
`公告`

### kab-curriculum-*
`课程`

### kab-member-*
`会员`

### kab-survey-*
`概论`

### kab-comment-*
`评论`

### kab-news-*
`新闻`
